# SRE-Zero v1 Bundle

This folder is a local release bundle for the first SRE-Zero public draft.

## Contents

- `paper/` - v1 technical report PDF.
- `jsons/` - copied evaluation JSON records from `notes/runs/`.
- `plots/` - copied plot assets from `notes/plots/`.
- `sre-zero-environment-grounded-evaluation-v1.md` - draft blog post announcing the v1 technical report.

## Draft Status

This is an early public draft. The current results are preliminary and are intended to validate the benchmark design, not to provide final model rankings.

## Publishing

The blog post is public and is ready to publish through the numbered blog folder workflow.
